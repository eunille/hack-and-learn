## 📖 How to Use This Repo  
Each lab/project is stored in its own folder under `/labs/` or `/projects/`.  
Every folder contains:  
- A **README.md** with documentation  
- Screenshots (if any)  
- Scripts or notes (if needed)  

Example structure:  
```
cybersecurity-labs/
│── labs/
│   └── blue-labs/
│   └── blue/
│       └── README.md
│   └── red-labs/
│── projects/
│── screenshots/
│── README.md   <-- (this file)
```
